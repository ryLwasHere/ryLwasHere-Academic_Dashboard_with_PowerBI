# 🎓 Academic Dashboard with Power BI

Dashboard interaktif untuk menganalisis kondisi sumber daya akademik perguruan tinggi di Indonesia menggunakan Power BI.

## 📊 Deskripsi Proyek

Proyek ini dikembangkan untuk memvisualisasikan dan mengeksplorasi data akademik perguruan tinggi, dengan fokus pada:

- Distribusi Jabatan Akademik Dosen
- Komposisi Usia Dosen
- Sebaran Profesor dan Lektor Kepala
- Analisis Regenerasi Akademik
- Risiko Krisis Kepemimpinan Akademik
- Insight berbasis data untuk pengambilan keputusan

## 🔍 Insight Utama

### Krisis Kepemimpinan Akademik

- Jumlah Profesor masih sangat terbatas.
- Mayoritas Profesor berada pada kelompok usia mendekati masa pensiun.
- Pipeline regenerasi Profesor belum memadai.
- Institusi berpotensi mengalami kekosongan kepemimpinan akademik dalam 5–10 tahun mendatang.

### Struktur Jabatan Akademik

- Sebagian besar dosen berada pada jenjang Asisten Ahli dan Lektor.
- Proporsi Lektor Kepala dan Profesor masih relatif rendah.

### Demografi Dosen

- Distribusi usia menunjukkan dominasi kelompok usia menengah hingga senior.
- Regenerasi tenaga akademik perlu dipercepat untuk menjaga keberlanjutan institusi.
